# 🎠 3D Rotating Carousel

A modern and interactive **3D Rotating Carousel** built with **HTML, CSS and JavaScript**. This project showcases a smooth infinite rotating carousel with realistic 3D perspective, responsive layout, and fluid animations, making it ideal for portfolios, product showcases, image galleries, and landing pages.

---

## 🚀 Features

- 🎠 **Realistic 3D rotating carousel**
- ✨ **Smooth CSS animations**
- 🖱️ **Mouse & touch interaction**
- 📱 **Fully responsive design**
- 🎨 **Modern glassmorphism-inspired UI**
- ⚡ **Lightweight & dependency-free**
- 🔄 **Infinite rotation**
- 🧩 **Easy to customize**
- 🌐 **Cross-browser compatible**
